# Pétanque Club Pépieux — Application PWA

## Contenu du dossier
- `index.html` — Page d'accueil de l'application
- `gestion.html` — Stocks & Événements
- `budget.html` — Budget du club
- `manifest.json` — Configuration PWA
- `sw.js` — Service Worker (fonctionnement hors ligne)
- `icon-192.png` / `icon-512.png` — Icônes de l'application

## Installation sur GitHub Pages (étapes)

### 1. Créer un compte GitHub
→ https://github.com/signup (gratuit)

### 2. Créer un nouveau dépôt
- Cliquer sur "New repository"
- Nom : `pepieux-app` (ou autre)
- Cocher "Public"
- Cliquer "Create repository"

### 3. Uploader les fichiers
- Cliquer "uploading an existing file"
- Glisser TOUS les fichiers de ce dossier
- Cliquer "Commit changes"

### 4. Activer GitHub Pages
- Aller dans Settings → Pages
- Source : "Deploy from branch"
- Branch : main / root
- Cliquer Save

### 5. Votre URL sera :
https://VOTRE-PSEUDO.github.io/pepieux-app/

### 6. Installer sur Android
- Ouvrir l'URL dans Chrome sur Android
- Menu ⋮ → "Ajouter à l'écran d'accueil"
- L'application s'installe comme une vraie appli !
